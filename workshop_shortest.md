# 🚀 1–2 Day “Math + Code + AI” Fun Bootcamp

**Goal:** Students enjoy learning math & coding, confidently explain their project, and learn how to self-learn using online tutorials + ChatGPT.

This curriculum is designed to be the **shortest possible high-impact workshop** that builds:

* Math confidence
* Coding foundations
* AI awareness
* Independent learning skills

---

# 📘 TABLE OF CONTENTS

1. Learning How to Learn Online (Meta Skill)
2. Math Reflection & Personal Challenge Selection
3. Coding Foundations (Game Thinking)
4. Build a Math Game (MakeCode – micro:bit Simulator)
5. Intro to AI & Machine Learning (Even vs Odd Classifier)
6. Circuits & Robotics (Tinkercad + micro:bit simulator)
7. Demo Day: Explain With Confidence & Joy

---

# ⏱ Recommended Format

**Option A:** 1 Full Day (6–7 hours)
**Option B:** 2 Half Days
**Option C:** 4 Short Sessions (2 hrs each)

TV/projector used for all demonstrations to avoid repeating instructions individually.

---

# 🧠 MODULE 1: How to Learn From an Online Tutorial (45 mins)

### Objective:

Students learn how to:

* Follow a tutorial step-by-step
* Pause & experiment
* Ask ChatGPT good questions
* Debug independently

### Live Demo (On TV)

Open:

* [https://makecode.microbit.org/](https://makecode.microbit.org/)
* [https://createai.microbit.org/](https://createai.microbit.org/)
* [https://www.tinkercad.com/learn/circuits/collection/circuits-basics](https://www.tinkercad.com/learn/circuits/collection/circuits-basics)

Show:

1. How to read instructions
2. How to copy example
3. How to modify example
4. How to ask ChatGPT:

   * “Explain this block in simple words”
   * “Why is my score not increasing?”
   * “How do I add a timer?”

### Mini Activity:

Students copy 1 simple MakeCode example and modify:

* Change text
* Change speed
* Change variable

✅ Success Outcome:
Students are not afraid of tutorials anymore.

# 🔢 MODULE 2: Math Self-Diagnosis (30 mins)

### Ask Students:

“What math topic is hard for you?”

Examples:

* Fractions
* Place value
* Decimals
* LCM/HCF
* Ratios
* Negative numbers
* Percentages

### Task:

Each student:

1. Writes 2–3 weak math topics.
2. Revises using ChatGPT.
3. If still confused → ask teacher/TA.

### Then:

Each student chooses **ONE topic** to turn into a game.

⚠ Each student can build multiple topics later.

# 🤖 MODULE 5: AI vs Logic – Even & Odd Classifier (1.5 hrs)

Platform:
[https://createai.microbit.org/](https://createai.microbit.org/)

## Project: Classify Handwritten Digits as Even or Odd

### Concept Teaching (On TV – 20 mins)

Explain:
Traditional logic:

```
number % 2 == 0
```

AI logic:
Computer learns from images instead of rules.

---

### Student Activity (Time Optimized)

Because of time:
Students only train:

* Digit 0 (Even)
* Digit 1 (Odd)

Using:
Mouse drawing.

Steps:

1. Create 2 classes: Even & Odd
2. Draw many 0 examples
3. Draw many 1 examples
4. Train model
5. Test prediction

### Key Lesson:

AI is good at:

* Finding patterns in images
* Not following math rules
* Learning from data
