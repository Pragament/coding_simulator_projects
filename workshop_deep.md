# 🚀 4-Week Deep Learning Program

**Math + Coding + AI + Circuits + Robotics**
Middle School (Grade 6–8) | 2 Sessions per Week | 2–2.5 hrs per session

**Program Vision:**
Students become joyful problem-solvers who can:

* Identify weak math topics
* Build math games using coding
* Train simple AI models
* Understand circuits & logic
* Learn independently using tutorials + ChatGPT
* Explain projects with confidence

---

# 📘 PROGRAM STRUCTURE OVERVIEW

## Week 1 — Learn How to Learn + Coding Foundations

## Week 2 — Math Game Development Mastery

## Week 3 — AI & Machine Learning (Even/Odd Classifier)

## Week 4 — Circuits, Robotics & Final Showcase

Platforms:

* **MakeCode micro:bit Simulator** → [https://makecode.microbit.org/](https://makecode.microbit.org/)

# 📅 WEEK 1 — Foundations: Think Like a Coder

## 🎯 Goals:

* Remove fear of coding
* Teach independent learning
* Introduce core programming concepts

---

## Session 1: Learning How to Learn (Meta Skill)

### Live Demo (TV once for all students)

Show:

* How to read an online tutorial
* How to pause and experiment
* How to debug
* How to ask ChatGPT effectively

### Teach Students to Ask:

* “Explain this like I am 11.”
* “Why is my score not increasing?”
* “What does this block do?”

### Activity:

Copy a simple MakeCode example.
Modify:

* Change text
* Change speed
* Add score variable

---

## Session 2: Core Coding Concepts

Using micro:bit simulator:

Students practice:

* Variables
* Random number generation
* If–Else
* Modulus operator (%)
* Comparison operators
* Events
* Input
* Output
* Loops
* Counters
* Boolean logic
* Functions
* Nested conditions

Mini Challenge:
Build a number guessing game.

---

# 📅 WEEK 2 — Math Game Builder

## 🎯 Goals:

* Connect math to coding
* Build personalized math games
* Add gaming mechanics

---

## Session 3: Math Diagnosis & Design

Students:

1. List 3 weak math topics.
2. Revise using ChatGPT.
3. Choose 1 main topic.

Examples:

* Fractions
* Place value
* LCM/HCF
* Percentages
* Ratios
* Negative numbers

### Game Planning Template:

* What is the math concept?
* What question will the game generate?
* How does player answer?
* How do you score?

---

## Session 4: Add Game Features

Students must include:

Core:

* Score counter
* Win/Lose condition
* Random question generation

Choose 3–5:

* Timer countdown ⏳
* Lives ❤️
* Difficulty modes
* Streak bonus
* Bonus points
* Sudden death
* Level progression
* Hidden Easter egg 🎉

By end of Week 2:
Each student has a playable math game.

---

# 📅 WEEK 3 — AI & Machine Learning

## 🎯 Goals:

* Understand difference between logic and AI
* Train a simple classifier
* Explain machine learning clearly

---

## Session 5: AI vs Logic

Explain:

Traditional Logic:

```
number % 2 == 0
```

AI:
Learns patterns from images.

Discuss:

* What is training data?
* What is a model?
* What is prediction?
* What is accuracy?

---

## Session 6: Build Even/Odd Classifier

Using:
CreateAI micro:bit

Time-efficient dataset:
Train only:

* Digit 0 (Even)
* Digit 1 (Odd)

Steps:

1. Create 2 classes
2. Draw many examples
3. Train model
4. Test new drawings

Discussion:
Why does it sometimes make mistakes?

Key Understanding:
AI is pattern recognition — not math reasoning.

---

# 📅 WEEK 4 — Circuits, Robotics & Showcase

## 🎯 Goals:

* Connect coding to physical computing
* Understand input/output systems
* Present confidently

---

## Session 7: Circuits & Robotics

Using Tinkercad:

Build:

* LED circuit
* Push button input
* Traffic light logic

Concept Mapping:

* Button → Input
* LED → Output
* If–Else → Decision
* Loop → Continuous behavior

Optional:
Connect game logic to LED behavior.

---

## Session 8: Demo Day + Reflection

Each student presents:

1. Their math challenge
2. Their game
3. Coding concepts used
4. AI classifier demo
5. Circuit explanation
6. One bug they solved
7. One thing they learned independently

Encourage:
Joy, storytelling, confidence.
