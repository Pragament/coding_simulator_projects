If student shows 3+ of these consistently:

☐ Avoids starting work
☐ Immediately asks for answers
☐ Fear of making mistakes
☐ Doesn’t test code
☐ Cannot explain own work

Action Plan:

☐ Pair with stronger peer
☐ Give smaller challenge
☐ Ask guiding questions only
☐ Provide confidence boost task
