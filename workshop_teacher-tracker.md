# 📊 Teacher Assessment & Grading Tracker

## 4-Week Math + Coding + AI + Circuits Program

Grade 6–8

This system is designed to measure:

* Concept understanding
* Independent learning ability
* Problem-solving mindset
* Creativity
* Confidence & communication

It balances **skills + growth + effort**, not just correctness.

# 📌 1️⃣ Math Understanding (20 Marks)

| Level      | Description                                   | Marks |
| ---------- | --------------------------------------------- | ----- |
| Beginner   | Basic math logic but limited understanding    | 8–10  |
| Developing | Random question generation + correct checking | 11–15 |
| Proficient | Multi-level difficulty + correct logic        | 16–18 |
| Advanced   | Creative math application + deep clarity      | 19–20 |

### Checklist:

☐ Chose appropriate math topic
☐ Questions are mathematically correct
☐ Can explain concept clearly
☐ Can solve sample question without help
